# /patent-search

> 연구 주제를 입력하면 AI가 WIPS ON 특허 검색식을 자동 설계하고,  
> 웹 대시보드에서 결과를 확인/편집할 수 있는 Claude Code 스킬입니다.

---

## 사용법

```
/patent-search 로보틱스
```

---

## 실행 흐름

| 단계 | 내용 |
|:---:|------|
| 1 | `/patent-search 주제` 입력 |
| 2 | Claude가 검색식 JSON 자동 생성 |
| 3 | 웹 대시보드 자동 실행 |
| 4 | 브라우저에서 결과 확인 및 편집 |
| 5 | 검색식 복사 → WIPS ON에서 검색 |

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 분류체계 설계 | 대-중-소분류 체계로 15~25개 소분류 자동 구성 |
| 한/영 키워드 | 소분류별 8~15개 동의어 자동 생성 |
| IPC 매칭 | 관련 IPC/CPC 코드 자동 추천 |
| 검색식 생성 | WIPS ON 신택스 기반 기본/정밀 검색식 |
| 웹 편집 | 키워드, IPC, 검색식을 브라우저에서 바로 수정 |
| 수정 추적 | AI 원본 대비 변경 사항 시각적 표시 |

## 폴더 구조

```
patent-search/
├── SKILL.md        Claude가 읽는 스킬 명세서 (수정 불필요)
├── README.md       이 파일
└── dashboard/      검색식 결과 웹 화면 (자동 실행)
    ├── app/        페이지 및 API
    ├── components/ UI 컴포넌트
    ├── lib/        유틸리티
    └── package.json
```

---

## 설치

이 저장소 URL을 Claude Code에 `이 스킬 설치해줘` 와 함께 붙여넣으면 됩니다.  
`dashboard/`의 `npm install`은 최초 실행 시 자동 처리됩니다.
